# 2026-06-04: Made the anydice pcb

**Total time spent: 2 hours**

anydice is a dice that can be set to any number and rolled for rng. it takes physical shaking movements via mpu6050, and u can set the max number from a button. today, i made the pcb,which is using a xiao esp32c3 as the brain of the pcb, mpu6050 for shaking detection, 1.3" oled for the numbers, and a generic switch. this i an idea i made myself, as i am a dnd dm. i also added a linear induction generator, which is a magnet moving from one fixed thick copper solenoid to another in a line, which generates electricity, exactly what u wud see in those shaking flashlight thingeys, which i thought were very cool. ill make a sphere with id 60mm, and add a weight to the bottom and a flat side. maybe i might make this into like a dropship product
